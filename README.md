**Narmavya** is a high-fidelity, cinematic e-commerce platform dedicated to showcasing and preserving the rich cultural heritage and artisanal craftsmanship of Madhya Pradesh, India. Built with a "Mobile-First" and "Editorial-First" design philosophy, it offers an immersive shopping experience that feels more like a digital museum than a traditional store.

---

## ✨ Key Features

- 🎞️ **Cinematic UI**: Powered by **GSAP** and **Framer Motion** for buttery-smooth animations and editorial layouts.
- 📦 **Bento Grid Architecture**: Modern, responsive layouts for product discovery and storytelling.
- 🔐 **Secure Authentication**: Integrated with **Firebase Auth** for seamless social and email logins.
- 💳 **Seamless Payments**: Robust checkout flow with integrated payment gateway simulation and order persistence.
- 📱 **Adaptive Responsiveness**: Optimized for every device, from ultra-wide monitors to mobile screens.
- 🛠️ **Admin Dashboard**: Comprehensive control panel for managing products, orders, and user reviews.
- 🧡 **Wishlist & Cart**: Persistent state management using **Zustand** and **React Query**.

---

## 🛠️ Tech Stack

### Frontend
- **Framework**: React 18 (Vite)
- **Styling**: Tailwind CSS
- **Animations**: GSAP (GreenSock) & Framer Motion
- **State Management**: Zustand
- **Data Fetching**: TanStack Query (React Query)
- **Authentication**: Firebase Client SDK

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB Atlas (Mongoose ODM)
- **Auth/Admin**: Firebase Admin SDK
- **Deployment**: Vercel (Serverless Functions)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB Atlas account
- Firebase Project

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/paladityax08/Narmavya.git
   cd Narmavya
   ```

2. **Install Dependencies**
   ```bash
   # Root directory (Frontend)
   npm install

   # Backend directory
   cd backend
   npm install
   ```
